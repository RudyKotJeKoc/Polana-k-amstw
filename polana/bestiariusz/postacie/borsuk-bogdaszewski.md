# Borsuk Bogdaszewski (Policja)

**Typ:** Bierny Obserwator Systemowy / Funkcjonariusz Biurokratyczny
**Powiązania:** Niebieska Karta (17.07.2021), Wiedźma Adamowska, System instytucjonalny

## Opis

Borsuk Bogdaszewski to zbiorowa postać reprezentująca **policję i system instytucjonalny**, który był wykorzystywany jako narzędzie w sporze rodzinnym na Polanie Kłamstw. Symbolizuje **bierność instytucji** — funkcjonariuszy, którzy wypełniają procedurę, ale nie widzą kontekstu, nie zadają trudnych pytań, nie kwestionują manipulacji.

Niebieska Karta została wszczęta 17.07.2021 — w kontekście sporu majątkowego (darowizna z 2017), ale Borsuk nie widzi tego kontekstu. Widzi tylko procedurę: formularz, zeznania, zapis. **Biurokracja zamiast wnikliwości.**

Borsuk (zwierzę) kopie nory, żyje pod ziemią, unika konfrontacji. Borsuk Bogdaszewski również unika — unika trudnych pytań, unika kwestionowania narracji, unika odpowiedzialności. Wypełnia formularz i odchodzi.

## Symbolika

**Borsuk jako zwierzę podziemne**
Żyje pod ziemią, unika światła. Borsuk Bogdaszewski unika trudnych prawd, unika kwestionowania, unika odpowiedzialności.

**Bierna obserwacja**
Widzi, ale nie interweniuje. Wypełnia procedurę, ale nie zadaje pytań.

**Niebieska Karta jako narzędzie manipulacji**
Instrument ochrony ofiar przemocy domowej — ale w tej historii stał się **bronią** w sporze majątkowym. Borsuk nie widzi tej instrumentalizacji.

**Biurokracja zamiast wnikliwości**
Formularz, zeznania, zapis. Procedura jest ważniejsza niż kontekst.

**Smerfy (policja biurokratyczna)**
W baśni policja nazwana "Smerfami" — mali, niebiescy, biurokratyczni. Wypełniają procedurę, ale nie widzą prawdy.

## Rola w Historii

**W głównej baśni "Polana Kłamstw":**

Borsuk Bogdaszewski nie jest antagonistą — jest **biernym narzędziem**, które zostało zinstrumentalizowane przez Wiedźmę. Jego rola to symbol **porażki systemu instytucjonalnego**, który nie potrafi zobaczyć poza procedurą.

**Kluczowe wydarzenia:**

1. **17.07.2021: Wszczęcie Niebieskiej Karty**
   - Niebieska Karta jako instrument ochrony ofiar przemocy
   - **Kontekst:** Spór majątkowy (darowizna z 2017), eskalacja konfliktu
   - Borsuk wypełnia formularz, nie widzi kontekstu
   - Procedura jest wykonana, ale prawda nie jest zbadana

2. **19.07.2021: Pełnomocnictwo Barbary (2 dni później!)**
   - Barbara uzyskuje pełnomocnictwo od Sylwestra
   - Niebieska Karta + Pełnomocnictwo = instrumentalizacja systemów
   - Borsuk nie widzi tej synchronizacji

3. **Bierna rola w procesie**
   - Policja jako źródło formalnych dokumentów
   - Zeznania, protokoły, raporty — bez krytycznej analizy
   - System staje się bronią Wiedźmy

**W kronice:**
Borsuk Bogdaszewski reprezentuje **bierność instytucji**, które mogą być zinstrumentalizowane przez manipulatorów. Jego rola to ostrzeżenie: **biurokracja bez wnikliwości czyni system narzędziem dla tych, którzy wiedzą jak go wykorzystać**.

## Cechy Charakterystyczne

- **Bierny obserwator** — widzi, ale nie interweniuje
- **Biurokrata** — wypełnia procedurę, nie zadaje pytań
- **Nie widzi kontekstu** — Niebieska Karta w sporze majątkowym
- **Unika trudnych pytań** — jak borsuk unika światła
- **Narzędzie, nie antagonista** — został zinstrumentalizowany
- **Smerf** — mały, niebieski, biurokratyczny
- **Symbol porażki instytucji** — system, który nie chroni, tylko wypełnia formularz
